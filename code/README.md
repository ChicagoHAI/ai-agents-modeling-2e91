# Cloned Repositories

## 1. Werewolf Arena
- **URL**: https://github.com/google/werewolf_arena
- **Location**: `code/werewolf_arena`
- **Purpose**: Evaluation framework for LLM social reasoning.
- **Key Features**:
  - Environment for LLM-based Werewolf.
  - Evaluation metrics for deception and deduction.

## 2. AIWolf-Python
- **URL**: https://github.com/AIWolfSharp/aiwolf-python
- **Location**: `code/aiwolf-python`
- **Purpose**: Python SDK for the AIWolf project (standard competition).
- **Key Features**:
  - Protocol-based communication (can be adapted for NLP).
  - Connection to AIWolf servers.

## 3. ChatArena-Werewolf
- **URL**: https://github.com/xuyuzhuang11/Werewolf
- **Location**: `code/ChatArena-Werewolf`
- **Purpose**: Implementation of Werewolf using ChatArena multi-agent framework.
- **Key Features**:
  - LLM-centric interaction.
  - Configurable roles and prompts.
