# Downloaded Datasets

## 1. LLMafia
- **Source**: HuggingFace (`niveck/LLMafia`)
- **Path**: `datasets/LLMafia`
- **Description**: Transcripts from 33 Mafia games with human and LLM players.
- **Type**: Text (JSON/CSV)

## 2. Werewolf-Among-Us
- **Source**: HuggingFace (`bolinlai/Werewolf-Among-Us`)
- **Path**: `datasets/Werewolf-Among-Us`
- **Description**: Multimodal dataset (video/transcript) of social deduction games.
- **Size**: ~9GB (includes video/audio features)

## Usage
These datasets are git-ignored. To use them, load directly from the paths above.
